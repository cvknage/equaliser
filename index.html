System-wide audio equalizer for macOS

Download for macOS

Features

64-Band Parametric EQ

Precise control over your audio with up to 64 bands of parametric equalization (PEQ).

System-Wide Processing

All audio from every app passes through the equalizer automatically.

Compare Mode

A/B testing to compare your EQ settings with the original sound in real-time.

Factory Presets

11 carefully crafted presets for music, movies, and more.

Light & Dark Mode

Automatically adapts to your macOS theme.

Installation

  1. Download

    Get the latest DMG from the link above.

  2. Install

    Drag Equaliser to your Applications folder.

  3. Launch

    Right-click and select Open on first launch to bypass Gatekeeper.

    Trouble launching?

    macOS Gatekeeper may block Equaliser because it's not notarized by Apple.

    If the app won't open, go to System Settings → Privacy & Security and click Open Anyway next to the security warning.

    I currently don't want to pay Apple $99/year to let the app through Gatekeeper.
    I may reconsider if it is requested.

Technical Specs

Band Count
Use 1 to 64 bands, add or remove bands as needed
Per-Band Controls
Each band has independent Frequency, Gain, Bandwidth/Q, and Filter Type
Filter Types
Bell, Low/High Pass, Low/High Shelf, Band Pass, Notch
Frequency Range
20 Hz – 20 kHz, fully adjustable per band
Sample Rate
Up to 768 kHz, supports high-resolution audio
Processing
32-bit float precision throughout the pipeline
Stereo Processing
Independent L/R channels, 64-band EQ per channel
REW Import
Import filter settings from Room EQ Wizard
